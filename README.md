# Evidencias GA3-220501093-AA3 - Programación y Algoritmia

Este repositorio contiene el desarrollo de las evidencias correspondientes a la actividad de aprendizaje **GA3-220501093-AA3** del programa de formación **Análisis y Desarrollo de Software (ADSO)** del SENA.

## Contenido del Repositorio

### 1. Evidencia 1: Bases Teóricas (Documentación)
Se encuentra detallada la investigación sobre:
- Diferencias entre lenguajes compilados e interpretados.
- Características principales de JavaScript.
- Tipos de datos primitivos.
- Operadores (Aritméticos, Lógicos, Comparación).

### 2. Evidencia 2: Resolución de Problemas Algorítmicos
Implementación de soluciones técnicas en JavaScript para los siguientes problemas:
- **Problema 1:** Cálculo de área y perímetro de figuras planas (Triángulo, Rectángulo, Cuadrado, Círculo).
- **Problema 2:** Análisis estadístico de un vector de edades (10 personas).
- **Problema 3:** Mezcla y ordenamiento de vectores ascendentes.

## Estructura del Proyecto

## Requisitos de Ejecución
Los scripts están diseñados para ser ejecutados en:
1. Consola del navegador (F12 -> Console).
2. Entornos de Node.js (requiere adaptaciones menores para entrada de datos).
